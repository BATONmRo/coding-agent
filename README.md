# Описание проекта

Этот проект представляет собой агент для автоматизации задач.

## Как это работает

Агент работает по следующему принципу: Issue → PR → Review → Iteration.

## Workflows

В проекте используются следующие workflow:
- workflow1.yml (триггер: push)
- workflow2.yml (триггер: pull_request)

## Secrets

Secrets используются для хранения конфиденциальной информации, такой как токены и пароли.

## Локальный запуск

Для локального запуска агента выполните следующие команды:
```bash
pip install code_agent
python -m code_agent.cli run
```

## Как проверить

Чтобы проверить работу агента, выполните следующие шаги:
1. Создайте issue.
2. Убедитесь, что issue был преобразован в PR.
3. Проведите ревью PR.
4. Завершите итерацию.
